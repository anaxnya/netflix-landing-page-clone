# Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS.

## Features

- Netflix-inspired UI
- Hero section with background overlay
- Navigation bar
- Email signup section
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- Flexbox

## Learning Outcomes

- Flexbox layouts
- Positioning and z-index
- Background overlays
- Responsive design principles
- CSS styling and animations

## Future Improvements
